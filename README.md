# 3D Connect4
- Observations
  - Shape: (4,6,7)
  - Values:
    - 0: Empty space
    - 1: Current player's piece
    - -1: Opponent's piece
- Actions
  - int [0, 41]
- Render
  - When initializing pass ``render_mode='human'`` to allow for game to be rendered

# To Run Games
```python
python3 test_models.py
```

You change which agent in the test_models.py file
